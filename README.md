## Overview

Numina helps you find workout partners and discover group fitness classes. Think ClassPass meets Meetup, with a focus on community, accountability, and fitness connections (not dating).

## Features

- User profiles with fitness preferences and availability
- Class discovery and filtering
- Partner matching (coming soon)
- Group coordination and messaging (coming soon)
- Ratings and feedback (coming soon)

## Technology Stack

- **Language**: Swift
- **UI**: SwiftUI
- **Min iOS**: 15.0, Target iOS: 18.0
- **DI**: Manual dependency injection (or Swinject)
- **Networking**: URLSession + Async/Await
- **Local DB**: SwiftData or Core Data
- **Image Loading**: AsyncImage / Kingfisher
- **Navigation**: NavigationStack
